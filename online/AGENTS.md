## SkillBar 安装与启动

1. 确保本机安装 Node.js `>= 20.9.0`。
2. 克隆仓库：`git clone git@github.com:Ch1ldKing/SkillBar.git`
3. 进入项目目录：`cd SkillBar`
4. 安装依赖：`npm install`
5. 启动开发环境：`npm run dev`
6. 在浏览器中打开：`http://localhost:3000`
7. 首次使用时，在页面中点击 `配置 Claude Token`，填入可用的 Anthropic API Key。

如需生产模式运行，可执行：`npm run build && npm run start`。
